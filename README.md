# dotfiles

Personal dotfiles for nvim, tmux, and bash.

## Structure

```
nvim/        — Neovim config (lazy.nvim, LSP, telescope, etc.)
tmux/        — tmux config with Dracula theme
bash/        — Custom bash functions and shell setup
install.sh   — Symlinks configs and sets up shell on a new machine
```

## Install

```bash
git clone git@github.com:shahanmny/dotfiles.git ~/dotfiles
cd ~/dotfiles && ./install.sh
```
